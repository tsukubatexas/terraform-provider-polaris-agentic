# Support

This repository is a public proof-of-concept for an autonomous Apache Polaris Terraform provider workflow.

## Best Places To Ask

- Open a GitHub issue for reproducible bugs.
- Open a GitHub discussion or issue for design questions.
- Use private vulnerability reporting for security issues.

## What To Include

- provider version or commit SHA
- Apache Polaris version
- Terraform version
- operating system and architecture
- minimal Terraform configuration
- sanitized logs

Do not include real credentials, access tokens, client secrets, Terraform state, or cloud account identifiers.
